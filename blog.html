How do you select an element by its ID in the DOM?

What are the differences between querySelector and querySelectorAll?

querySelector returns the first element that matches a specified CSS selector.

querySelectorAll returns a static NodeList of all elements that match the specified CSS selector.

How do you add a new element to the DOM?

You can add a new element to the DOM using the createElement() and appendChild()
